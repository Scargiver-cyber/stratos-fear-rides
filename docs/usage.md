# Usage

## Running from source

```bash
python main.py
```

Follow the on-screen main menu to:

1. Run the full simulation (planning + booking + launch)
2. Run quick status mode (no booking or launch)
3. Change terminal display mode
4. Exit

## Installing as a package

```bash
pip install .
stratosfear
```

You can also install from the GitHub checkout:

```bash
pip install git+https://github.com/<your-user>/stratosfear-rides.git@v1.0.0
```
