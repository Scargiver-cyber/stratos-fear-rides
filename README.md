# Stratos-FEAR Rides

**Affordable Adrenaline-Forward Space Sightseeing**

At Stratos-FEAR Rides, our mission is to turn the thrill of space exploration into heart-pounding, unforgettable adventures. We bring the excitement of the cosmos closer than ever—one fearless ride at a time.

## About This Project

This is a Python class project simulating a space tourism agency. The project demonstrates core Python concepts including variables, lists, tuples, functions, loops, and user input handling.

## How to Run

```bash
# Run Day 1 - Agency Overview
python3 space_agency_day1.py

# Run Day 2 - Operations Report & Booking System
python3 space_agency_day2.py
```

## Features

### Day 1
- Agency information and mission statement
- Spacecraft fleet with seat capacities
- Crew rosters (captains, copilots, flight attendants, flight ops)
- Available missions and destinations
- Basic calculations (total seats, fuel requirements)

### Day 2
- Functions for fuel calculation and crew assignments
- Loops to generate reports
- Interactive customer booking system
- Input validation for group sizes

## Skills Demonstrated

- Python variables and data types
- Lists and tuples
- Functions with parameters and return values
- For loops and iteration
- User input and validation
- Conditional statements (if/else)
